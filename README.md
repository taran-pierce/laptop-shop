# 🖥️ Gaming Laptop Deal Tracker

This is a simple Python script that searches the web for deals on:

- ASUS ROG Strix SCAR 18 (RTX 5070 Ti)
- ASUS ROG Strix G18 (RTX 5070)

It collects pricing data daily, saves it to a CSV file, and alerts you when prices hit your target deal thresholds.

---

## 🚀 Features

- 🔎 Scrapes Google Shopping results
- 💾 Saves results to a CSV file (appends daily)
- 📉 Tracks price history over time
- 🔥 Flags "good deals" based on target prices
- ⚡ Lightweight and easy to run
